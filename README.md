# 🐦 Chennai Pet's World - Website

A modern, mobile-first website for **Chennai Pet's World**, an exotic pet shop located in Kolathur, Chennai, specializing in exotic birds, dogs, and premium pet accessories.

## 🎨 Design

### Color Palette
- **Primary (Teal):** `#009688` - Trust, nature, calm
- **Accent (Yellow):** `#FFC107` - Energy, warmth, exotic birds
- **CTA (Orange):** `#FF5722` - Action, urgency
- **Dark:** `#1a1a2e` - Text, footer

### Typography
- **Headings:** Poppins (Bold)
- **Body:** Inter (Regular)

## ✨ Features

### Core Functionality
- ✅ Mobile-first responsive design
- ✅ Floating WhatsApp button (click to chat)
- ✅ Click-to-call phone button
- ✅ Inquiry forms with WhatsApp redirect
- ✅ Google Maps integration
- ✅ AOS scroll animations

### Page Features
- **Home:** Hero section, featured pets, testimonials, inquiry form
- **About:** Brand story, stats (500+ families, 50+ varieties), values
- **Pets:** Filterable catalog (Birds, Dogs, Accessories, Food)
- **Gallery:** Masonry grid with lightbox viewer
- **Reviews:** Customer testimonials with rating breakdown
- **Contact:** Contact form, FAQ accordion, store info

## 🚀 Getting Started

1. **Clone or download** the project files
2. **Open** `index.html` in any modern browser
3. No build process required - works out of the box!

### For Development
```bash
# Optional: Use a local server for better experience
npx serve .
# or
python -m http.server 8000
```

## 📱 Responsive Breakpoints

- **Mobile:** < 768px
- **Tablet:** 768px - 1024px
- **Desktop:** > 1024px
